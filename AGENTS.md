# AGENTS.md

## 项目名称

职业天赋挖掘

## 项目目标

这是一个职业天赋测评与 AI 报告生成网站。用户完成问卷后，系统根据 14 个职业维度得分生成不少于 5000 中文字的职业天赋挖掘报告，并引导用户继续使用深度职业智能体对话。

## 技术栈

* Next.js
* TypeScript
* Tailwind CSS
* Supabase
* DeepSeek V4 API
* Recharts 或 SVG/CSS 图表
* lucide-react 图标

## 核心约束

1. 不要破坏现有问卷和评分逻辑。
2. 14 个职业维度必须保持一致：

   * 收入目标感
   * 成长模式
   * 认同感来源
   * 社交消耗度
   * 边界清晰度
   * 认同逻辑
   * 压力节奏
   * 权力与执行
   * 思维颗粒度
   * 信息处理
   * 驱动逻辑
   * 反馈周期
   * 环境空间依赖
   * 道德阈值
3. 首页视觉风格为小红书爆款轻测评风。
4. 移动端优先。
5. 报告不能玄学化、不能医学诊断、不能绝对化承诺。
6. 完整报告渲染后的中文总字数不得少于 5000。
7. 不要每次从零生成整份报告，要使用本地分析资产库 + AI 个性化融合。
8. 未配置 DEEPSEEK_API_KEY 时，必须可以使用 mock report 本地开发。
9. paid=false 展示预览，paid=true 展示完整报告。
10. 所有新增代码必须尽量类型安全、结构清晰、可维护。

## 运行命令

优先使用：

* npm install
* npm run dev
* npm run build
* npm run lint

如果项目中没有某些脚本，请先检查 package.json，再选择可用命令。
